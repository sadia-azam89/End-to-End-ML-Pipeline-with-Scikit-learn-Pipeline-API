# End-to-End-ML-Pipeline-with-Scikit-learn-Pipeline-API
This project demonstrates how to build a complete Machine Learning pipeline using Scikit-learn Pipeline API.  It covers the full ML lifecycle:  Data preprocessing Feature engineering Model building Hyperparameter tuning Model evaluation  🎯 Objective: Predict whether a customer will churn or not.
